## 进制转换通用模板

1. 将P进制数x转换为十进制数y
~~~
int x, p;
	int y = 0, product = 1;
	while (!x) {
		y += (x % 10) * product;
		x /= 10;
		product *= p;
	}
~~~

2. 将十进制数y转换为Q进制数z
~~~
  int Q , y;
  int q[40];
	int index = 0;
	do {
		q[index++] = y % Q;
		y /= Q;
	} while (!y);
~~~
注意：
+ q中的逆序才是Q进制数的正确顺序

+ do while结构比while结构好在可以处理y=0的情况

+ 如果Q≥10，可以在转化中添一步，也可以在输出时处理，视情况而定
