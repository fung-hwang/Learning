5. **二叉树交换左右结点**
	+ 动态树
		以**先/后序遍历**的方式遍历树时交换左右孩子结点  
		**中序遍历是不可以的**
	~~~
	void invert(int root) {
	if (root == -1) {
		return;
	}
	invert(Node[root].lchild);
	invert(Node[root].rchild);
	swap(Node[root].lchild, Node[root].rchild);
	}
	~~~
	+ 静态树
	直接扫描数组一一交换左右孩子结点即可
	~~~
	void invert() {
		for (int i = 0; i < n; i++) {
			swap(Node[i].lchild, Node[i].rchild);
		}
	}
	~~~
